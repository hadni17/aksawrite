# Cara Install
buka terminal/cmd, masuk ke direktori dimana source code ini tersimpan, lalu jalankan perintah berikut
```
$ pip install -r requirements.txt
```

# Cara Menjalankan
untuk menjalankan program ini, jalankan perintah berikut
```
$ python main.py
```
